# Node.js Blueprint

## Dependencies

- [Node.js](http://nodejs.org/)

## Installation

Clone Git repo, `cd` into directory, and run `npm install`.

## Usage

**Running the development environment**  

```shell
npm run dev
```
Then visit `http://localhost:7171`
